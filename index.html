CLASTool by usm-data-analysis

A fork of the CLASTool package with a new Makefile

ClasTool is a ROOT based package for analyzing CLAS data. It consists of a set of C++ classes in a number of libraries which you can use to build your own CLAS Analysis program or script. Visit the page of the original project to read more about it.

This is a fork of the original source code. It has a new non-recursive Makefile, instead of the old recursive one. For a discussion about it, read this.

Dependencies

You need a proper ROOT installation, and some CLAS libraries. Read installation instructions for more details.
